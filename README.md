## Install & Run

git clone https://github.com/pldtn/pencileasyapp.git
cd pencileasyapp
npm install
npm run dev

## Libraries

- [html2canvas] - Turns DOM into Canvas.
- [jsPDF] - To generate PDF from images.
- [cypress] - Testing Library
- [serve] - Start local server
